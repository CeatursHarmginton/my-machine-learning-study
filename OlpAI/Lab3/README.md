# Linear Regression Activity Pack

This folder contains a complete activity sequence for teaching linear regression holistically.

## Contents

- `LINEAR_REGRESSION_ACTIVITY_GUIDE.md`: detailed instructor guide with activities, math, visuals, and discussion prompts.
- `datasets/`: synthetic CSV datasets for each activity.
- `notebooks/linear_regression_activity_starter.ipynb`: Python starter notebook with plotting and modeling code.
- `requirements.txt`: suggested Python packages.

## Suggested Python setup

```bash
pip install -r requirements.txt
jupyter notebook notebooks/linear_regression_activity_starter.ipynb
```

## Notes

All datasets are synthetic and designed for teaching. They include deliberately constructed cases such as nonlinearity, heteroscedasticity, outliers, multicollinearity, saturation, and overfitting.
